Listing Studio

Privacy Policy

Listing Studio is a desktop application designed to help users manage their own eBay listings.

Information Collection

Listing Studio does not collect, store, sell, or share personal information from eBay users.

Data Storage

All listing information is stored locally on the user's computer. Listing Studio does not operate an external database for storing user information.

eBay Authorization

When authorized by the user through eBay OAuth, Listing Studio accesses only the user's own eBay listing information for the features requested by the user.

Third Parties

Listing Studio does not sell or disclose personal information to third parties.

Policy Updates

This Privacy Policy may be updated as the application evolves.
